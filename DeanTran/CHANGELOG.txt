[V1.2.0] - OCR Translation Foundation
---------------------
* Bổ sung OCR phát hiện chữ trong ảnh (hỗ trợ trích text từ điểm ảnh và dịch block).
* Hỗ trợ textbox overlay cho ảnh trong file PowerPoint/Word/Excel.
* Tái sử dụng cơ chế bảo vệ date/time và token chuyên ngành trong block ảnh để skip qua các token không cần dịch.

* Tăng logging chuyên sâu và fail-safe cho ảnh lỗi (error blocks không làm đổ bể nguyên file).

[V1.1.0] - 2026-03-07
---------------------
* Cơ chế bảo vệ ngày/giờ và mã Token (Skip Translation).
* Quản lý API Key (Key Pool), Round-robin, Auto-rotate. Cho phép nhập nhiều key cùng lúc.
* Giao diện UI tùy chỉnh giới hạn (Max Items/Chars per Batch, Timeout, Retries, Circuit Breaker).
* Phân Quyền & Session Timeout (Admin, Operator, User). Admin tự đăng xuất sau 30 phút.
* Checkpoint Job, Lưu `_stop` Logs (Tự động lưu file .json chứa checkpoint khi dừng / huỷ dịch giữa chừng).
* Sửa lỗi (bug fix) và tối ưu độ ổn định.

[V1.0.0] - 2026-03-06
---------------------
Phiên bản phát hành đầu tiên (First Official Release).
* Tính năng Dịch Excel:
  - Dịch theo Batch siêu nhanh.
  - Hỗ trợ giữ nguyên định dạng, giữ nguyên ảnh sau khi tạo sheet bản sao (ví dụ: Sheet1_Vi).
* Tính năng Dịch PowerPoint:
  - Dịch toàn bộ Slide, gom nhóm theo đoạn (paragraph) để giữ ngữ cảnh.
  - Fix lỗi crash ứng dụng sau khi dịch hoàn tất.
* Tính năng Dịch Word (đã hỗ trợ nền tảng).
* Tính năng Thuật Ngữ (Glossary) & Bảo vệ Định dạng bằng Regex.
* Cấu hình an toàn:
  - Toàn bộ file Logs, Config, Output được trích xuất ra thư mục riêng biệt giúp người dùng dễ dàng theo dõi.
  - Không ghi đè mất file gốc (_Vi(1), _Vi(2)).
